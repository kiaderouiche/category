Tutorials

We highly recommend working through a tutorial as a quick and easy way to learn how to write tests.

Starting the tutorial

Our tutorials use the Eclipse 'cheat sheets'. Follow these steps to open the cheat sheets.

  1. Select: 'Help -> Cheat Sheets' from the menu
  2. Select the cheat sheet you want to do
  3. Click OK

We recommend that you start with the First Steps cheat sheet, as the other cheat sheets assume that you have already understood Test Case and Test Suite creation, as well as object mapping.


Once the cheat sheet is open, maximize the client to the full screen size and start working! Everything is explained step by step.
