Context sensitive help

The following points give you an overview of the documentation available as context-sensitive help.


  1. The user manual contains information on the concepts, how to complete various tasks with the tool, and information on the installed example projects.
  2. The reference manual contains information on the supported components and actions, and on regular expressions, language codes, shortcuts etc.
